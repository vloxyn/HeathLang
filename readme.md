### What is Heath?

Heath is a custom translator that translates it's own syntax over to **LUA.**

### Features

- Simple and very memorizable syntax (maybe)
- Translation only takes 1 milliesecond!

### Usage

Load the provided loadstring under for Heath with any executor that you use, make sure it is at least good enough as in being able to run Heath.

```lua
loadstring(game:HttpGet('https://raw.githubusercontent.com/vloxyn/HeathLang/main/lang.lua', true))()
```
